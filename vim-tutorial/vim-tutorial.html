VIM

VIM is a editor using without mouse!
: => w (write) | q (quit) | q! (quit without save)

Move using keys

  • K: Up
  • J: Down
  • L: Right
  • H: Left

Modes

Visual Mode => ctrl + V

That mode allows you to select a block of caracters or code, using the command showed before.

Insert Mode => I (to blink point) || A (to insert avance one character on blink point) || O (to break a line and enter)

That mode allows to write the document.

Typing commands

  • yy => copy a line
  • shift + p => paste above the copied
  • p => paste below the copied element

  • d => Delete
    • dd => delete line

  • gg => top of archive in vim
  • shift + g => bottom of archive in vim

  • u => undo changes in vim
  • ctrl + R => re-undo changes in vim

  • 0 => start in vim
  • $ => end in vim

  • [ ({) => skip code blocks: Up
  • ] (}) => skip code blocks: Down

  • x + gg => Acess a x numbers line of the archive
  • x + move keys => Move a x numbers characters of the line
  • x + move keys => Move a x numbers characters of the line
